# AI Memory System — 提交部署包

这是一个纯前端 Vite + React 演示，不需要 API key、数据库或后端服务。

## 最快发布

把 `dist/` 里的全部文件上传到任意静态托管平台的发布目录即可。

- 体验主页：`/`
- 项目说明：`/about.html`

## 从源码构建

```bash
npm install
npm run build
```

然后上传新生成的 `dist/`。

## 当前体验范围

预置案例 → 留下 → 先放着 → Back 收件 → 回到一个曾经的我 → 补一句。

这是受控的概念验证：使用本地 mock 和浏览器 localStorage；不包含真实 LLM、账号、云端数据库或图片导入。
